# Gitbook book-tool Button Plugin 

This plugin adds a book-tool button to your GitBook. All three GitBook themes (White, Sepia, Night) are supported.

## Usage

Add the plugin to your `book.json`:

```
{
	"plugins" : [ "book-tool" ]
}		
```
